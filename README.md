# 🕷️ SPIDEY AI Assistant

> Your Friendly Neighborhood AI Assistant — Now with Voice! 🎤🗣️

A powerful desktop AI assistant with multi-provider AI, persistent memory, semantic search, auto-memory detection, and full voice conversation.

## 🚀 Features

### ✅ Completed
- 🤖 **Multi-AI Provider** — Groq, OpenAI, DeepSeek, Ollama
- 💬 **Interactive Chat** — Terminal-based AI chat
- 🧠 **SQLite Database** — Structured data storage
- 🔍 **ChromaDB Vectors** — Semantic search by meaning
- 🧠 **Auto Memory** — Detects name, age, city automatically
- 📝 **Notes System** — Personal notes with categories
- 🔍 **Smart Search** — Exact + semantic combined
- ⚙️ **Settings System** — Configurable everything
- 📋 **Logging System** — App, chat, error logs
- 🎤 **Voice Input** — Whisper speech-to-text
- 🗣️ **Voice Output** — Edge TTS (Jenny voice)
- 🕷️ **Spidey Beta** — Full voice conversation mode
- 🌍 **Multi-language** — English, Urdu, Hindi voices
- 🔇 **Auto-stop** — Records until you stop talking

### 🔜 Coming Soon
- 💻 Computer Control (PyAutoGUI)
- 🌐 Web Search & Internet Tools
- 🤖 AI Agent Brain
- 🖥️ Desktop GUI (PySide6)
