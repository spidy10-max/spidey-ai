# Spidey Tech Stack Research — Day 6

## 1. FastAPI
- Python ka modern web framework
- Bohot fast hai (async support)
- Automatic API docs banta hai (Swagger UI)
- Spidey mein: Backend server banayenge jahan AI brain se baat hogi

## 2. SQLite
- File-based database (no server needed)
- Python mein built-in support hai (sqlite3 module)
- Spidey mein: Chat history, user preferences, personal notes save karenge

## 3. ChromaDB
- Vector database for AI embeddings
- Semantic search allow karta hai (meaning-based search)
- Spidey mein: "Find chats about X" type intelligent search karenge

## 4. Whisper (OpenAI)
- Speech-to-text model (audio to text)
- Multiple languages support including Urdu
- Offline chalta hai — no internet needed
- Spidey mein: User ki voice sunke text mein convert karega

## 5. Piper TTS
- Text-to-speech engine (text to audio)
- Offline, fast aur lightweight
- Multiple voice models available
- Spidey mein: AI ka jawab bolke sunayega

## 6. PyAutoGUI
- Desktop automation library
- Mouse move, click, type, screenshot le sakta hai
- Spidey mein: "Open Chrome", "Take screenshot" jaise commands execute karega

## 7. PySide6
- Qt-based GUI framework for Python
- Cross-platform (Windows, Mac, Linux)
- Modern aur professional looking UI bana sakte hain
- Spidey mein: Beautiful desktop app window banayenge with chat interface

## 8. Ollama
- Local LLM runner (no internet needed)
- Llama, Mistral, Qwen jaise models locally run karta hai
- Spidey mein: Offline AI brain as fallback when no internet
