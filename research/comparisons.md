# Spidey — Library Comparisons

## 1. Web Framework Comparison

| Feature | FastAPI | Flask | Django |
|---------|---------|-------|--------|
| Speed | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| Async Support | ✅ Built-in | ❌ No | ⚠️ Limited |
| Learning Curve | Easy | Easy | Hard |
| Auto API Docs | ✅ Yes | ❌ Manual | ❌ Manual |
| Best For | APIs & AI | Simple apps | Full websites |

**✅ Winner: FastAPI** — Async support, fast, auto docs, perfect for AI backend

## 2. Database Comparison

| Feature | SQLite | PostgreSQL | MongoDB |
|---------|--------|------------|---------|
| Setup | Zero config | Server needed | Server needed |
| Speed | Fast | Fast | Fast |
| Size | Tiny (single file) | Heavy | Medium |
| Python Support | Built-in | psycopg2 | pymongo |
| Best For | Desktop apps | Web apps | JSON/NoSQL data |

**✅ Winner: SQLite** — No server needed, built into Python, perfect for desktop app
